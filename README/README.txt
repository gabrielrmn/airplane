# COMO JOGAR (teclas e funcionalidades) O JOGO : TRABALHO PRÁTICO 2 . 

	-tecla '1' --> câmera de cima do avião.
	-tecla '2' --> câmera third person ( de trás do avião).
	-tecla '3' --> câmera dentro da cabine do avião (first person).
	-mouse     --> movimenta a posicao de visao do avião.
	-tecla 'w' --> avião traça um caminho reto.
	-tecla 'a' --> avião se direciona para o lado esquerdo.
	-tecla 's' --> avião se direciona para trás
	-tecla 'd' --> avião se direciona para o lado direito
	-tecla 'l' --> liga/desliga a luz.
	-tecla 'n' --> liga/desliga o fog.
	-tecla 'p' --> se utilizada no local de pouso ( indicado por um H ) pousa/decola o avião.
	-tecla 'Esc' --> fecha o game.
